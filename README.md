# Rozhi Coffee Plan

پروژه Laravel + Blade + Vanilla JS + Tailwind CSS برای یک دعوت ساده، تمیز و آماده آپلود روی سرور.

## تغییرات این نسخه

- مشکل چندبار پخش شدن آهنگ رفع شد؛ فقط یک پلیر اصلی باقی مانده است.
- پلیر موزیک بین چند تب/صفحه قفل دارد؛ اگر آهنگ در یک جای دیگر در حال پخش باشد، جای دیگر دوباره اجرا نمی‌شود.
- اجرای خودکار اضافی که با اولین کلیک صفحه آهنگ را دوباره پخش می‌کرد حذف شد.
- مسیر عکس جایگزین گربه مشکی آماده شد: `public/images/rozhin.jpg`
- همه جاهایی که قبلاً `images/black-kitten.svg` استفاده شده بود به مسیر جدید `ROZHI_CAT_IMAGE` وصل شد.
- UI برای وب و موبایل، مخصوصاً iPhone 12 به بعد، بهتر شد: safe-area، دکمه‌های لمسی بزرگ‌تر، music dock موبایل، فاصله‌ها و تایپوگرافی واکنش‌گرا.
- خروجی Vite داخل `public/build` قابل تولید است و پروژه برای آپلود روی سرور آماده است.

## جایگذاری عکس Rozhin

فایل عکس روی سیستم شما اینجاست:

```txt
H:\rozhin.jpg
```

چون این مسیر فقط روی کامپیوتر شما قابل دسترسی است، داخل پروژه این مسیر استاندارد ساخته شده است:

```txt
public/images/rozhin.jpg
```

قبل از آپلود نهایی، فایل `H:\rozhin.jpg` را در مسیر بالا داخل پروژه کپی کنید. تنظیم آن در `.env` هم آماده است:

```env
ROZHI_CAT_IMAGE=images/rozhin.jpg
```

اگر عکس هنوز کپی نشده باشد، برای جلوگیری از خراب دیده شدن صفحه، تصویر قدیمی `black-kitten.svg` به عنوان fallback نمایش داده می‌شود.

## آهنگ پس‌زمینه

فایل فعلی موزیک در این مسیر خوانده می‌شود:

```txt
public/audio/lana-let-the-light-in.mp3
```

پلیر پایین صفحه فقط با دکمه `♫` اجرا می‌شود و دیگر audio دوم یا اسکریپت کلیک خودکار وجود ندارد.

## اجرا روی لوکال

```bash
php -S 127.0.0.1:8000 -t public
```

یا:

```bash
php artisan serve
```

## نصب از صفر / توسعه

```bash
cp .env.example .env
composer install
npm install
php artisan key:generate
php artisan migrate:fresh --seed
npm run build
php artisan serve
```

## نکته سازگاری

برای اینکه روی هاست‌های مینیمال هم صفحه بالا بیاید، پروژه در صورت نبودن درایور SQLite از دیتای fallback داخل `PageController` استفاده می‌کند.
